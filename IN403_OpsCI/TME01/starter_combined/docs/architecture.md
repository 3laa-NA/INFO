
# Architecture cible du projet FilmFinder (à terme)

- Frontend : React + Vite (application web)
- Backend  : API REST (FastAPI ou Node/Express)
- Base de données : PostgreSQL
- Conteneurisation : Docker + Docker Compose
- Orchestration : Kubernetes (kind ou k3s)

Dans le TME1, vous ne codez pas encore l'application : 
vous prenez en main Linux, le réseau et les scripts.
