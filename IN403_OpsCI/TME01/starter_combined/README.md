
# Starter Pack — TME1 OpsCI

Ce dossier sera utilisé pour les premiers TME du projet "FilmFinder".

Contenu :
- data/ : données d'exemple sur des films
- docs/ : documentation d'architecture (à lire)
- scripts/ : scripts shell à compléter/utiliser
- config/ : fichiers de configuration d'exemple

Pour le TME1 :
- téléchargez et décompressez ce dossier
- explorez-le avec les commandes vues en cours (ls, tree, find, cat, etc.)
